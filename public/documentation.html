Introduction

Movie API will provide users with access to information about different movies, directors and genres. Users will be able to sign up, update their personal information and create a list of their favorite movies.

Dependencies

Local Dependencies

  • Body Parser
  • Express
  • Lodash
  • UUID

Development Dependencies

  • ESLint
  • Morgan
  • Nodemon

API Endpoints

Request Method URL Response
GET /movies List of all movies
GET /movies/:title Data about a movie by title
GET /movies/genres/:genreType Data about a genre by type
GET /movies/directors/:directorName Data about a director by name
POST /users Allow new users to register
PUT /users/:id Allow users to update their user id
POST /users/:id/:movieTitle Allow users to add a movie to their favorites
DELETE /users/:id/:movieTitle Allow users to remove a movie from their favorites
DELETE /users/:id Allow existing users to deregister
