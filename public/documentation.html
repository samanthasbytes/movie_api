Introduction

Movie API will provide users with access to information about different movies, directors and genres. Users will be able to sign up, update their personal information and create a list of their favorite movies.

Dependencies

Local Dependencies

  • Body Parser
  • Express
  • Lodash
  • UUID

Development Dependencies

  • ESLint
  • Morgan
  • Nodemon

API Endpoints

Business Logic HTTP Method URL Endpoints Path Variables Request Body Data Format Response Body Data Format
Return a list of all movies to the user GET /movies None None A JSON object holding data about all movies.
Return data (description, genre, director) about a single movie by title to the user GET /movies/:title :title None A JSON object holding data about a single movie.
A JSON object holding data about a single movie. GET /movies/genres/:genreType :genreType None A JSON object holding data about a single genre.
Return data about a director (bio, birth year, death year) by name GET /movies/directors/:directorName :directorName None A JSON object holding data about a single director.
Allow new users to register POST /users None A JSON object that requires the name of the new user. For example:
{ "name": "Kim" } 		A JSON object holding data about the user. The object includes the name specified in the request body and a user ID generated by the uuid.v4(); function. For example:
{ "name": "Kim", "id": "52a2eff2-83b6-4581-8baa-dd649667f827" }
Allow users to update their name PUT /users/:id :id A JSON object that requires the name of the user. For example:
{ "name": "Jessica" } 		A JSON object holding data about the user. The change in name will be reflected in the response. For example, assuming the users name was previously Kim, then changed to Jessica:
{ name: "Jessica", "id": "52a2eff2-83b6-4581-8baa-dd649667f827", "favoriteMovies": [] }
Allow users to add a movie to their favorites POST /users/:id/:movieTitle :id
:movieTitle 		None A string indicating that the movie title has been added to the user’s favorites. For example:
Inception has been added to Kim's favorites.
Allow users to remove a movie from their favorites DELETE /users/:id/:movieTitle :id
:movieTitle 		None A string indicating that the movie title has been removed from the user’s favorites. For example:
Inception has been removed from Kim's favorites.
Allow existing users to deregister DELETE /users/:id :id None A string indicating that the user has been removed. Specifies the name of the user. For example:
Ben has been removed.
